   Follows

2. Technologies
   MongoDB

3. Installation
   Clone the repository
   git clone https://github.com/narendravs/Social-mern-app.git

4. Install dependencies
   cd social-mern-app

   Install server dependencies
   cd api
   npm install
   npm start

   Install client dependencies
   cd client
   npm install
   npm start
